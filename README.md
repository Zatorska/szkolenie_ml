# Wprowadzenie do Machine Learning na przykładzie drzewa decyzyjnego

Repozytorium powstało na potrzeby szkolenia dla firmy **Alten** pt.  
## **„Wprowadzenie do Machine Learning na przykładzie drzewa decyzyjnego”**, 
### _praktycznych warsztatów w Pythonie skierowanych do osób pracujących z danymi_.

Szkolenie w przystępny sposób wprowadza podstawowe pojęcia Machine Learning oraz problem klasyfikacji, pokazując **jak działa model ML „od środka”** na przykładzie drzewa decyzyjnego. Uczestnicy krok po kroku budują i oceniają prosty model w Pythonie, ucząc się interpretacji wyników, unikania overfittingu oraz dobrych praktyk na starcie pracy z ML.

Materiał ma charakter **praktyczny i warsztatowy** – jego celem jest zrozumienie logiki działania modeli oraz zdobycie solidnych podstaw do dalszej nauki i samodzielnych eksperymentów.
